Step 1: Data Preparation

Let's start by preparing the data for our ANOVA analysis.


# Python code for data cleaning
import pandas as pd

# Load the raw dataset
df = pd.read_excel('ca-lab-results-2023-05-30.xlsx')

# Filter to keep only relevant columns
filtered_df = df[['producer', 'product_type', 'total_thc']]

# Drop rows with missing values in these columns
cleaned_df = filtered_df.dropna(subset=['producer', 'product_type', 'total_thc'])

# Save the cleaned data for further analysis
cleaned_df.to_excel('cleaned_ca_lab_results.xlsx', index=False)

Step 2: ANOVA Analysis

Now, let's perform the ANOVA analysis on our cleaned data.


# Python code for ANOVA
import scipy.stats as stats

# Load the cleaned data
cleaned_df = pd.read_excel('cleaned_ca_lab_results.xlsx')

# ANOVA by Producer
anova_producer = stats.f_oneway(*(cleaned_df['total_thc'][cleaned_df['producer'] == producer] for producer in cleaned_df['producer'].unique()))

# ANOVA by Product Type
anova_product_type = stats.f_oneway(*(cleaned_df['total_thc'][cleaned_df['product_type'] == product_type] for product_type in cleaned_df['product_type'].unique()))

Step 3: Interpretation

Let's interpret the results of our ANOVA analysis.

  • By Producer: The p-value was 0.082, greater than the 0.05 threshold, indicating no significant difference in THC levels across producers.
  • By Product Type: The p-value was extremely close to zero (1.96 x 10-15), indicating a significant difference in THC levels across product types.
